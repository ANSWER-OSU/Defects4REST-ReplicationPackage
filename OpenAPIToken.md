## **How to Generate an OpenAI API Token**

**Step 1:** Go to [OpenAI](https://platform.openai.com/) and **sign in**. If you don’t have an account, click **Sign Up** to create one.

![OpenAPI Screenshot](/images/openAPI1.png)

**Step 2:** Once logged in, click on your **profile icon** in the top-right corner and select **View API Keys** (or go directly to [https://platform.openai.com/account/api-keys](https://platform.openai.com/account/api-keys)).

**Step 3:** Click **Create new secret key**.

![OpenAPI Screenshot](/images/openAPI2.png)

**Step 4:** Fill out the information and copy the generated token immediately. You **won’t be able to view it again** after leaving the page.

![OpenAPI Screenshot](/images/openAPI3.png)

**Step 5:** Save the token securely. You will use it in Step 1.5
